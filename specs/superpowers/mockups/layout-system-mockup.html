Per-Page Overrides

Individual posts and pages can override the default layout from their editor.

How it works: When editing any post or page, the Layout metabox in the editor sidebar lets the author pick a specific layout for that content. If no override is set, the default layout for that content type (configured above) is used.

Additionally, each page can toggle Hide Header and Hide Footer independently — useful for landing pages, funnels, or embedded content.
